
# Blog Guide

## Commands to Install

Installed apps
```
git
visual studio code
nodejs
```

command line
```
npm install hexo-cli -g
hexo init yusabudun
cd yusabudun
npm install
```
To test the site
```
hexo server
```

## Editing the Site

- Blog posts and pages are in source folder
- For configuration edit _config.yml
- For theme configuration edit themes/hexo-casper/_config.yml

## Deploy
```
hexo clean
hexo generate
hexo deploy
```
